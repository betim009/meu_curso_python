# Exercício 08 - Herança com funcionários

Crie uma classe base `Funcionario` e uma classe filha `Vendedor`.

## Requisitos

### Funcionario

- Atributos: `nome` e `salario`.
- Método `calcular_pagamento`.

### Vendedor

- Deve herdar de `Funcionario`.
- Atributo extra: `comissao`.
- Deve sobrescrever `calcular_pagamento`.

## Situação real

Um vendedor é um funcionário, mas seu pagamento tem uma regra diferente.
