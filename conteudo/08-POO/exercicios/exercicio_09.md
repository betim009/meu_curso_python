# Exercício 09 - Formas de pagamento

Crie três classes:

- `PagamentoCartao`
- `PagamentoPix`
- `PagamentoBoleto`

## Requisitos

- Todas devem ter um método `processar(valor)`.
- Cada classe deve retornar uma mensagem diferente.
- Crie uma lista com as três formas de pagamento.
- Percorra a lista chamando `processar(100)`.

## Situação real

Objetos diferentes podem responder ao mesmo método. Essa é a ideia básica de polimorfismo.
