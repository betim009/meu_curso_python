# Exercício 05 - Produto com estoque

Crie uma classe `Produto`.

## Requisitos

- Atributos: `nome`, `_preco` e `_estoque`.
- Crie `adicionar_estoque(quantidade)`.
- Crie `remover_estoque(quantidade)`.
- Crie `calcular_total_estoque`.
- Não permita estoque negativo.

## Situação real

Uma venda só pode remover produtos se houver quantidade suficiente.
