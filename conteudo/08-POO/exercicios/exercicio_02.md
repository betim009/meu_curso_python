# Exercício 02 - Produto com alteração de preço

Crie uma classe `Produto`.

## Requisitos

- Atributos: `nome` e `_preco`.
- Crie um método `alterar_preco(novo_preco)`.
- O preço só pode ser alterado se for maior que zero.
- Crie um método `consultar_preco`.
- Crie `__str__` para exibir o produto.

## Situação real

Um sistema não deve permitir que um produto tenha preço negativo ou igual a zero.
