# Exercício 05 - Conta bancária

Crie uma classe `ContaBancaria`.

## Requisitos

- A classe deve ter os atributos `titular` e `saldo`.
- Crie um método `depositar(valor)`.
- Crie um método `exibir_saldo`.
- Crie uma conta, faça um depósito e exiba o saldo final.

## Exemplo esperado

```text
Titular: Rafael Alves
Saldo: R$ 1250.00
```
