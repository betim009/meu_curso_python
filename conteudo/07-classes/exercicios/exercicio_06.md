# Exercício 06 - Produto com validação

Crie uma classe `Produto` com validação simples.

## Requisitos

- A classe deve ter `nome`, `preco` e `quantidade`.
- Se `preco` for menor que zero, guarde `0`.
- Se `quantidade` for menor que zero, guarde `0`.
- Crie um método `valor_total`.
- Crie um método `exibir_dados`.

## Situação real

Um sistema de estoque não deve aceitar preço ou quantidade negativa.
