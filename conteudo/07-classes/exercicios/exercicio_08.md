# Exercício 08 - Cliente com status

Crie uma classe `Cliente`.

## Requisitos

- A classe deve ter `nome`, `email` e `ativo`.
- O cliente deve começar ativo.
- Crie um método `desativar`.
- Crie um método `ativar`.
- Crie um método `exibir_status`.

## Situação real

Sistemas de cadastro geralmente não apagam clientes. Eles marcam o cadastro como ativo ou inativo.
