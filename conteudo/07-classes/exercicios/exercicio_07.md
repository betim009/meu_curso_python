# Exercício 07 - Funcionário com aumento

Crie uma classe `Funcionario`.

## Requisitos

- A classe deve ter `nome`, `cargo` e `salario`.
- Crie um método `aplicar_aumento(percentual)`.
- O aumento deve alterar o salário do funcionário.
- Crie um método `exibir_dados`.
- Teste aplicando um aumento de 10%.

## Situação real

Um sistema de RH precisa atualizar salários quando a empresa concede reajustes.
