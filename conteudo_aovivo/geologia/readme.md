Nesta primeira aula, é muito importante tirar da cabeça da aluna a ideia de que programar é algo distante ou impossível. O foco aqui é acolher, contextualizar e mostrar utilidade.

## Aula 2 — Condições e repetições aplicadas a dados

### Objetivo da aula

Ensinar como o Python pode tomar decisões e repetir ações automaticamente.

### Conteúdos

- `if`
- `else`
- `for`
- comparação de valores
- interpretação de resultados

### Abordagem didática

Aqui o ideal é mostrar que o Python consegue analisar informação e agir com base em uma regra.

Isso é muito útil para situações como:

- identificar valores altos ou baixos
- separar amostras por faixa
- marcar resultados suspeitos
- classificar informações

### Exemplo

```python
amostras = [10, 45, 70, 15]

for valor in amostras:
    if valor > 30:
        print("Concentração alta")
    else:
        print("Concentração baixa ou moderada")
```

### O que explicar detalhadamente

Explique com bastante calma:

- o `for` percorre item por item da lista
- `valor` é uma variável temporária
- `if` verifica uma condição
- `>` significa “maior que”
- o Python executa um bloco diferente dependendo do resultado da condição

### Atividade sugerida

Criar uma lista de valores de concentração e pedir para a aluna mostrar mensagens diferentes para:

- valores altos
- valores médios
- valores baixos

### Ganho pedagógico

Essa aula é excelente para começar a desenvolver pensamento lógico sem cair em exercícios sem sentido.

## Aula 3 — Funções e organização de pequenas tarefas

### Objetivo da aula

Mostrar como criar pequenos blocos de código reutilizáveis.

### Conteúdos

- o que é função
- `def`
- parâmetros
- retorno
- reaproveitamento de código

### Abordagem didática

Explique que uma função é uma forma de ensinar o Python a executar uma tarefa específica sempre que necessário.

Esse assunto é importante porque ajuda a aluna a:

- organizar melhor o código
- evitar repetição
- deixar a solução mais clara
- criar pequenas ferramentas próprias

### Exemplo

```python
def classificar_concentracao(valor):
    if valor >= 50:
        return "alta"
    elif valor >= 20:
        return "media"
    else:
        return "baixa"

print(classificar_concentracao(70))
```

### O que explicar

- `def` é usado para criar função
- `classificar_concentracao` é o nome da função
- `valor` é o dado que entra na função
- `return` devolve um resultado
- a função pode ser reutilizada várias vezes

### Atividade sugerida

Pedir para a aluna criar uma função que receba um valor e diga se a amostra está em nível baixo, médio ou alto.

## Aula 4 — Introdução ao Pandas e leitura de arquivos CSV

### Objetivo da aula

Mostrar como trabalhar com dados tabulares usando uma das bibliotecas mais importantes do Python.

### Conteúdos

- o que é biblioteca
- o que é `pandas`
- `import pandas as pd`
- leitura de CSV
- `head()`
- noção de tabela
- linhas e colunas

### Abordagem didática

Aqui a aluna começa a sentir que está entrando em um terreno muito mais próximo da realidade profissional, porque muitos dados de trabalho aparecem em formatos como:

### O que explicar com riqueza de detalhes

- `dados` passa a representar a tabela lida
- `head()` mostra as primeiras linhas

### O que também precisa ser explicado

A aluna precisa entender com clareza o que é:

- linha
- coluna
- nome de coluna
- valor de célula
- tabela de dados

## Aula 5 — Seleção, filtro e estatística básica com Pandas

### Objetivo da aula

Ensinar a aluna a extrair informações relevantes de uma base de dados.

### Conteúdos

- selecionar colunas
- filtrar linhas
- média
- máximo
- mínimo
- contagem

### Exemplo

```python
print(dados["concentracao"].mean())
print(dados[dados["concentracao"] > 30])
```

### Abordagem didática

Aqui a aluna começa a perceber o poder real do Python para análise.

Mostre que ela pode responder perguntas como:

- qual a média de concentração?
- quais amostras têm valor acima de certo limite?
- quais tipos de rocha aparecem mais?
- quais locais possuem mais registros?

### O que explicar

Explique que:

- `dados["concentracao"]` seleciona uma coluna
- `.mean()` calcula a média
- `dados[dados["concentracao"] > 30]` cria um filtro
- esse filtro devolve apenas as linhas desejadas

### Atividade sugerida

Pedir para a aluna:

- descobrir a média da profundidade
- filtrar amostras com concentração alta
- mostrar apenas os dados de um tipo específico de rocha

## Aula 6 — Tratamento e limpeza de dados

### Objetivo da aula

Ensinar a lidar com dados incompletos, errados ou bagunçados.

### Conteúdos

- valores ausentes
- `isna()`
- `dropna()`
- `fillna()`
- padronização simples
- cuidados com dados reais

### Abordagem didática

Essa aula é muito valiosa porque dados reais quase nunca vêm perfeitos.

A aluna precisa entender desde cedo que trabalhar com dados não é apenas fazer gráfico bonito ou calcular média. Muitas vezes, o primeiro trabalho é organizar a bagunça.

### Exemplo

```python
print(dados.isna().sum())

dados_limpos = dados.dropna()
```

### O que explicar

- `isna()` identifica valores vazios
- `sum()` ajuda a contar quantos estão faltando
- `dropna()` remove linhas com valores ausentes
- limpar dados é uma etapa essencial antes da análise

### Atividade sugerida

Usar uma tabela com alguns campos vazios e pedir para a aluna:

- identificar os vazios
- decidir como tratar
- comparar o antes e o depois
