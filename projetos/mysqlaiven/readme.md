# Projeto MySQL Aiven

## 1) Criar ambiente virtual (venv)

```bash
python3 -m venv .venv
```

## 2) Ativar o ambiente virtual

### macOS/Linux
```bash
source .venv/bin/activate
```

### Windows (PowerShell)
```powershell

```
